**Trackers Evaluation.ipynb contains:** <br/> 
Evaluation of RGB and Thermal videos as per metrics chosen as:<br/> 
RSTE: Root Mean Square Tracking Eroor<br/> 
IOU: Intersection over Union<br/> 
Precision <br/> 
Recall<br/> 
F-Measure<br/> 
Robustness<br/> 
Trackers used here:<br/> 
MIL<br/> 
TLD<br/> 
BOOSTING<br/> 
MEDIANFLOW<br/> 
CSRT<br/>
